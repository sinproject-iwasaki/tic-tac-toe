# Tic Tac Toe

Tic Tac Toe game built with [Rust](https://www.rust-lang.org/) and [Bevy Engine](https://bevyengine.org/).

## Demo

Play it here: [GitHub Pages](https://sinproject-iwasaki.github.io/tic-tac-toe/)

## Quick Start

Install dependencies and build:

```sh
cargo build
```

Run:

```sh
cargo run
```
